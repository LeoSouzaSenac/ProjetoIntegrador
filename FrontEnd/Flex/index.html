Guia Completo do Flexbox

Aprenda como funciona o Flexbox com exemplos práticos.

O que é Flexbox?

O Flexbox é um modelo de layout unidimensional que oferece uma maneira de distribuir o espaço entre os itens de um contêiner e alinhar esses itens de maneira flexível e previsível.

Propriedades do Contêiner Flex

As principais propriedades que você pode aplicar a um contêiner flex são:

  • display: flex; - Define um contêiner como flexível.
  • flex-direction: - Define a direção dos itens (row, row-reverse, column, column-reverse).
  • justify-content: - Alinha os itens ao longo do eixo principal (flex-start, flex-end, center, space-between, space-around).
  • align-items: - Alinha os itens ao longo do eixo transversal (flex-start, flex-end, center, stretch, baseline).
  • flex-wrap: - Define se os itens devem quebrar ou não em várias linhas (nowrap, wrap, wrap-reverse).

flex-direction

A propriedade flex-direction define a direção dos itens dentro do contêiner flexível. As opções são:

  • row (padrão)
  • row-reverse
  • column
  • column-reverse
Item 1
Item 2
Item 3
.flex-container {
    display: flex;
    flex-direction: row;
}

justify-content

A propriedade justify-content alinha os itens ao longo do eixo principal do contêiner. As opções são:

  • flex-start (padrão)
  • flex-end
  • center
  • space-between
  • space-around
  • space-evenly
Item 1
Item 2
Item 3
.flex-container {
    display: flex;
    justify-content: space-between;
}

align-items

A propriedade align-items alinha os itens ao longo do eixo transversal do contêiner. As opções são:

  • stretch (padrão)
  • flex-start
  • flex-end
  • center
  • baseline
Item 1
Item 2
Item 3
.flex-container {
    display: flex;
    align-items: center;
    height: 150px;
}

flex-wrap

A propriedade flex-wrap define se os itens devem quebrar em várias linhas, se necessário. As opções são:

  • nowrap (padrão)
  • wrap
  • wrap-reverse
